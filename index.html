fishtocli

Bash-скрипт для прогноза клёва прямо в терминале

Реальная погода с open-meteo, анализ перепадов давления и персональные вердикты для каждой рыбы.

Не гадай — проверяй

GitHub Установка Блог и гайды

Новое в версии 1.3

  • Свои локации — введи любые координаты, скрипт покажет прогноз
  • Сохранение в конфиг — добавленные локации появляются в меню (~/.config/fish2cli/locations.conf)
  • График давления — за последние 5 уникальных дней (от 800 до 1040 гПа)
  • Кастом вместо Салавата — теперь 3) Кастом для ручного ввода

Возможности

  • 16+ локаций — от Астрахани до Красноярска + свои
  • Реальная погода через open-meteo (без API-ключей)
  • Умный анализ давления — скрипт помнит вчерашние показания
  • 7 видов рыб со своим характером: щука, судак, окунь, сазан, лещ, карась, сом
  • Советы по приманкам и тактике
  • Учёт времени суток — ночные хищники не дремлют
  • Цветной вывод в терминале
  • График давления за последние 5 дней

Установка

git clone https://github.com/nespase/fish2cli
cd fish2cli
chmod +x fish2cli.sh

Зависимости

Скрипту нужны: curl, python3, bc

# Ubuntu/Debian
sudo apt install curl python3 bc

# Arch
sudo pacman -S curl python bc

# macOS
brew install curl python bc

# Windows
# установите WSL: https://learn.microsoft.com/ru-ru/windows/wsl/install

Запуск

./fish2cli.sh
Для своих локаций: выбери пункт 3) Кастом, введи координаты. После прогноза скрипт предложит сохранить локацию — она появится в меню при следующих запусках.

Логика работы

РыбаУсловия для активности
ЩукаПадение давления или пасмурно → "Мамка охотится!"
СудакВысокое давление (>1014) + ночь → ждет блесны
ОкуньВысокое давление (>1012) + ясно → дает жару
СазанТепло (>15°C) + южный ветер → режим пылесоса
ЛещСтабильное давление → ждет прикормки
КарасьТепло (>12°C) → активен, холодно → грустит
СомНочь → зверь, день → спит

Давление кэшируется в /tmp/ — скрипт помнит прошлый запуск и реагирует на перепады.
